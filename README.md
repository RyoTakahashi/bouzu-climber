# BOUZU CLIMBER 〜坊主への道〜

スマホブラウザ向けの1ファイル完結型タップゲーム。次々とやってくるお客さんの髪を刈り、切った髪の毛でどれだけ高いヘアタワーを築けるかを競う。

---

## ゲームの目的

お客さんの髪を4回タップして坊主にし、切り取った髪を積み上げて「ヘアタワー」の高さ（cm）を最大化する。

## 客の種類

### ノーマル客（通常）
- 開始から30秒間は全員ノーマル
- 髪ゾーンをタップするだけでOK
- 4回タップで坊主完成

### リクエスト客（💬）
- 「左は残して」「右は残して」「前髪だけ残して」「耳周りNG」の4種
- NGゾーンが赤い点滅オーバーレイで表示される
- NGゾーンをタップするとその場でゲームオーバー

### フェイント客（🧢）
- 帽子をかぶった怪しい客
- 「やっぱり…どうしようかな」と吹き出しが出る
- **正解操作:** 帽子を下方向にスワイプしてかぶせる（30px以上・下方向）
- **NG操作:** 髪・顔・耳をタップするとゲームオーバー

---

## 操作方法

| 操作 | 対象 | 効果 |
|------|------|------|
| タップ（髪ゾーン） | 通常・リクエスト客 | 髪を1回刈る。+2〜5cm 獲得 |
| タップ（顔・耳ゾーン） | 全客種 | 怒られてゲームオーバー |
| 下スワイプ（帽子） | フェイント客 | 帽子をかぶせてクリア |
| タップ（髪・顔・耳） | フェイント客 | 即ゲームオーバー |

---

## ゲームロジック

### バリカン進捗
- 1人につき4回タップで坊主完成
- 完成後、次のお客さんへ自動移行（1.2秒のアニメーション後）

### ヘアタワー
- タップするたびに切った髪（2〜5cm）がタワーに積み上がる
- タワーは右パネルのキャンバスにリアルタイム描画
- 自己ベスト更新時は「🏆NEW!」フラッシュ

### 次客タイマー
- 一定間隔で待機列に客が追加される
- 初期インターバル: 4000ms
- 経過時間に応じて加速: `speed = min(5, 1 + 経過秒 / 10)`（最大5倍速）
- 待機列が10人に達するとゲームオーバー

### 客の出現確率（30秒経過後）
| 種類 | 確率 |
|------|------|
| フェイント客 | 18% |
| リクエスト客 | 32% |
| ノーマル客 | 50% |
