# BOUZU CLIMBER 〜坊主への道〜

スマホブラウザ向けの1ファイル完結型タップゲーム。次々とやってくるお客さんの髪を刈り、切った髪の毛でどれだけ高いヘアタワーを築けるかを競う。

---

## ゲームの目的

お客さんの髪を4回タップして坊主にし、切り取った髪を積み上げて「ヘアタワー」の高さ（cm）を最大化する。

## チュートリアル

初回起動時に4枚スライドのチュートリアルが表示される。ヘッダーの「？」ボタンからいつでも再表示可能（表示中はゲーム一時停止）。

| スライド | 内容 |
|---------|------|
| 1 | ✂️ 髪をタップして刈る基本操作 |
| 2 | 🚫 顔・耳はタップ禁止 |
| 3 | 💬 リクエスト客のNGゾーン |
| 4 | 🧢 フェイント客の帽子スワイプ |

---

## 客の種類

### ノーマル客（通常）
- 開始から30秒間は全員ノーマル
- 髪ゾーンをタップするだけでOK
- 4回タップで坊主完成

### リクエスト客（💬）
- 「左は残して」「右は残して」「前髪だけ残して」「耳周りNG」の4種
- NGゾーンが赤い点滅オーバーレイで表示される
- NGゾーンをタップするとその場でゲームオーバー

### フェイント客（🧢）
- 帽子をかぶった怪しい客
- 「やっぱり…どうしようかな」と吹き出しが出る
- **正解操作:** 帽子を下方向にスワイプしてかぶせる（30px以上・下方向）
- **NG操作:** 髪・顔・耳をタップするとゲームオーバー
- バリアント（後述）は付かない

---

## バリアント（20秒経過後）

20秒以降、フェイント客以外にランダムでバリアントが付与される。髪を刈れる位置が変わり、難易度が上がる。

| バッジ | 種類 | 変化 |
|-------|------|------|
| ↙️ | 首かしげ左 | キャラが左に22°傾く → 髪ゾーンが左にずれる |
| ↘️ | 首かしげ右 | キャラが右に22°傾く → 髪ゾーンが右にずれる |
| 📏 | 高身長 | キャラが縦に伸びる → 髪ゾーンが上にずれる |

待機列のミニキャラにも同じ傾き・バッジが表示され、来る前に種類を確認できる。

---

## 操作方法

| 操作 | 対象 | 効果 |
|------|------|------|
| タップ（髪ゾーン） | 通常・リクエスト客 | 髪を1回刈る。+2〜5cm 獲得 |
| タップ（顔・耳ゾーン） | 全客種 | 怒られてゲームオーバー |
| 下スワイプ（帽子） | フェイント客 | 帽子をかぶせてクリア |
| タップ（髪・顔・耳） | フェイント客 | 即ゲームオーバー |

## ゲームロジック

### バリカン進捗
- 1人につき4回タップで坊主完成
- 完成後0.7秒のアニメーション（「次のお客様〜！」コール＋スライドイン）を経て次客へ移行
- アニメーション中もタイマーは進み続ける

### ヘアタワー
- タップするたびに切った髪（2〜5cm）がタワーに積み上がる
- タワーは右パネルのキャンバスにリアルタイム描画
- 自己ベスト更新時は「🏆NEW!」フラッシュ

### 次客タイマー
- 一定間隔で待機列に客が追加される
- 初期インターバル: 4000ms（= 1倍速）
- 待機列が10人に達するとゲームオーバー

#### 速度カーブ
| 経過時間 | 速度倍率 |
|---------|---------|
| 0秒 | 1.0倍 |
| 10秒 | 1.5倍 |
| 10〜20秒 | 1.5倍（固定） |
| 20秒 | 1.8倍 |
| 30秒 | 1.9倍 |
| 40秒 | 2.0倍 |
| 以降 | 10秒ごと +0.1倍 |

### 客の出現確率

**0〜29秒:** ノーマル客のみ

**30秒以降:**
| 種類 | 確率 |
|------|------|
| フェイント客 | 18% |
| リクエスト客 | 32% |
| ノーマル客 | 50% |

**バリアント付与（20秒以降・フェイント以外）:**
| バリアント | 確率 |
|-----------|------|
| 首かしげ左 | 28% |
| 首かしげ右 | 24% |
| 高身長 | 16% |
| なし | 32% |

## 技術スタック

- **HTML / CSS / Vanilla JS** 1ファイル完結
- Canvas API（ヘアタワー描画）
- SVG（キャラクター・ミニキャラ描画）
- `SVGElement.getScreenCTM()` によるタップ座標変換（CSS transform・preserveAspectRatio を自動考慮）
- Google Fonts（Dela Gothic One / Noto Sans JP）
- `requestAnimationFrame` ゲームループ
- Touch Events 対応（スマホブラウザ動作）
